Приложение не запустилось

Скорее всего файл открыт двойным кликом (адрес начинается с file://). Браузер тогда не загружает модули JavaScript.

Сделайте так:

  1. Откройте папку проекта в терминале и выполните: npm install затем npm run dev
  2. Перейдите по адресу, который покажет Vite (обычно http://localhost:5173)
  3. Или откройте собранный файл dist/index.html через «Live Server» / любой локальный HTTP-сервер
